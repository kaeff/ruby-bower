# Ruby::Bower

A ruby wrapper for [Bower](https://github.com/twitter/bower).

## Installation

Add this line to your application's Gemfile:

    gem 'ruby-bower'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby-bower
